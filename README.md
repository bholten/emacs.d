Minor additions to Steve Purcell's excellent emacs config:

https://github.com/purcell/emacs.d

Please see that repo for any issues.

Additions beyond Mr. Purcell's config:

- Minor theme and org-mode tweaks
- Pipenv for Python
- rvm for Ruby
- R/Emacs Speaks Statistics
- Standard ML
- Scheme/Racket
- Scala
- More C/C++ goodies (on-going, completely unstable, beware)
- Crystal language support
- Golang
- Ocaml
- Idris
- Elixir
- Nim

All these additions are generally vanilla and could use some
improvement.
